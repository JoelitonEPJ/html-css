# html-css
Curso de HTML5 e CSS3

Este repositório guarda todos os meus exercícios e desafios desenvolvidos durante o **Curso de HTML5 e CSS3** ministrado pelo professor Gustavo Guanabara, do [Curso em Vídeo](https://www.cursoemvideo.com/). 

O objetivo principal aqui é construir uma base sólida no desenvolvimento web front-end, entendendo a semântica da web e a estilização em cascata, evoluindo desde os conceitos básicos até a criação de layouts responsivos.
